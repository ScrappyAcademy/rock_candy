# RockCandy

Providing sugary syntax to help crystalize your test/spec structure.


## Installation

Add this line to your application's Gemfile:

```ruby
group :test do
  gem 'rock_candy'
end
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rock_candy
